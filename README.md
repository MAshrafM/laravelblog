# Laravel Blog
Custom Laravel 5.6 Blog with Bootstrap 4 and MYSQL.  

## Containing
- User Authentication
- Members Dashboard.
- Simple Crud for Posts
- CKeditor. 
- Custom Design with Bootstrap.
- Image Uploads.

## Future Aditions
- Categories and Tags
- Emails
- Ratting System
